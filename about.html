About the Project

This project explores the relationship between biodiversity and economic metrics by combining Wikidata biodiversity indicators with macroeconomic data. It surfaces continent and country-level summaries for endemic species, endangered species, GDP, and population in an interactive, client-side visualization.

Data sources

All metrics are retrieved from Wikidata using SPARQL queries against the QLever endpoint and are rendered in the browser using D3. Data is cached locally for fast comparisons.

How to use

Start at the interactive map to view continent rollups. Click a continent to drill down to individual countries and compare endemic vs. endangered species alongside GDP and population figures.
