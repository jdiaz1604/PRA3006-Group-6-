This prototype explores how endemic and endangered species relate to macroeconomic indicators using live Wikidata SPARQL queries.

If you have feedback, ideas for new indicators, or notice a data issue in Wikidata, feel free to reach out or open an issue in the project repository.

You can also:

  • Extend the code with new charts (e.g., CO₂ emissions, protected areas).
  • Connect it to your own SPARQL / QLever queries.
  • Embed the map into teaching material about biodiversity & economics.
